## Business Problem
The objective of this project was to provide decision-makers with clear visibility into operational performance and customer activity using KPIs and dashboards.

## Approach

- Built data warehouse models to support analytics
- Developed interactive Power BI dashboards
- Integrated dashboards into a web application
- Applied machine learning models for advanced analytics

## KPIs & Insights
- Operational performance indicators
- Usage trends and peak activity analysis
- Identification of anomalies and inefficiencies

## Tools
Power BI | SQL | Python | Talend | PostgreSQL | Flask
